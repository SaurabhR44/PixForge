## ğŸš€ Getting Started

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SaurabhR44/PixForge.git
   cd PixForge
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   cd frontend && npm install
   cd ..
   ```

3. **Start the Development Forge**:
   ```bash
   npm run dev
   ```
   *The backend runs on port 9000, and the frontend on port 5173.*

### Deployment to Vercel

PixForge is pre-configured for Vercel. Simply:
1. Push your code to a GitHub repository.
2. Import the project into Vercel.
3. Vercel will auto-detect the `vercel.json` configuration and deploy the full-stack app.
